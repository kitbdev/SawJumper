# SawJumper
### by kitbdev  
[Itch.io page](https://kitbdev.itch.io/sawjumper)

* Unity
* Blender
* [JFXR](https://jfxr.frozenfractal.com/)
* VS Code
* Git

Game controls (keyboard / gamepad):  
mouse / rstick- look  
wasd / lstick- move  
space / south button - dash jump, climb  
shift  / north button - center camera
